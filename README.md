# 인터랙티브 뉴스 템플릿

충청투데이 기획기사에 반복 사용할 수 있도록 만든 HTML/CSS/JavaScript 템플릿입니다.

## 가장 빠른 사용법

1. `content.js`를 연다.
2. `hero`, `metrics`, `sections`, `closing`의 문구와 수치를 바꾼다.
3. `assets` 폴더에 기사 사진을 넣는다.
4. `content.js`의 이미지 경로를 새 파일명으로 수정한다.
5. `index.html`을 브라우저에서 연다.

## 파일 구성

- `index.html`: 페이지 뼈대
- `styles.css`: 디자인·반응형
- `script.js`: 메뉴, 카운트업, 스크롤 노출, 그래프 애니메이션
- `content.js`: 기사별로 수정할 데이터
- `assets/`: 사진과 영상 썸네일

## 섹션 유형

### bars
막대그래프형 데이터에 사용합니다.

### ranking
순위, 과제, 단계형 데이터에 사용합니다.

### cards
쟁점, 사례, 제언을 카드로 나눌 때 사용합니다.

### photo
전체 화면에 가까운 대형 현장 사진 구간입니다.

## 배포

### 닷홈 FTP
1. 폴더 안 파일을 서버의 새 디렉터리에 업로드합니다.
2. 예: `/AI_interactive/new_story/`
3. 브라우저에서 해당 주소의 `index.html`을 확인합니다.

### 주의 사항
- 예시 수치를 그대로 보도에 사용하지 마십시오.
- 사진 용량은 가급적 장당 500KB 이하 WebP로 줄이는 편이 좋습니다.
- 한글 파일명보다 영문 소문자와 하이픈을 권장합니다.
- 외부 기사 링크는 `content.js`의 `closing.links`에서 교체합니다.
